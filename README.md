# ✨ Premium Quote Generator

A modern, minimalist, and premium Quote Generator built with a sleek glassmorphic UI. Generate inspirational and funny quotes with a single click and share them instantly with your network.

![Preview](https://raw.githubusercontent.com/rajjitlai/Quote_Generator/main/assets/preview.png)

## 🚀 Features

- **Glassmorphic Design**: A premium interface featuring background blurs, frosted glass effects, and ambient glow.
- **Dynamic Quotes**: Instantly generate high-quality quotes from various categories.
- **Micro-Animations**: Smooth fade-in transitions and interactive button effects.
- **Category Selection**: Choose between "Inspirational" and "Funny" quotes.
- **Social Sharing**: Built-in functionality to share your favorite quotes on Facebook and X (Twitter).
- **Responsive Layout**: Optimized for all screen sizes, from mobile to desktop.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom vanilla styling with glassmorphism, HSL colors, and CSS variables.
- **JavaScript (ES6+)**: Dynamic quote generation and event handling.
- **Google Fonts**: Leveraging 'Outfit' for a modern typographic feel.
- **FontAwesome**: High-quality iconography for controls and sharing.
